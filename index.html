Week 5 Homework: Trivia Game

This game is running right now so read fast. The timer gives you 12 seconds to answer each question. The counter doesn't appear until only 11 seconds left, thereby increasing the difficulty of the game.* If you don't click on an answer, the "unanswered" number gets a +1. If you guess incorrectly, "Wrong guesses" is incremented. You can guess as many times as you want per question. Want extra points? Keep clicking the correct answer during the 4 second timeout between questions!* If the number of correct answers reaches 5 before the sum of unanswered and wrong answers, you win.
* This is a feature not a bug.
